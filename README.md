# Customer-Trends-Data-Analysis-Project
This repository contains an end-to-end data analytics project focused on analyzing customer trends and behavior using PostgreSQL, Python, and Power BI.

### Project Overview

The goal of this project is to extract, clean, analyze, and visualize customer data to uncover meaningful patterns and insights that support data-driven decision making.

### Tools & Technologies

* **PostgreSQL** – Data extraction and querying
* **Python** – Data cleaning, transformation, and exploratory data analysis
* **Power BI** – Interactive dashboard creation and visualization

### Key Work Done

* Cleaned and preprocessed raw customer data
* Performed exploratory data analysis to identify trends and patterns
* Analyzed customer behavior and key business metrics
* Built an interactive Power BI dashboard for clear insight presentation

### Key Learnings

* Importance of data quality and preprocessing
* Using SQL and Python together for efficient analysis
* Converting analytical findings into business-friendly dashboards

### Repository Structure

* `PostgreSQL/` – Queries used for data extraction
* `Python/` – Data cleaning and analysis scripts
* `PowerBI/` – Dashboard file and visuals
* `Dataset/` – Raw and processed datasets

### Future Improvements

* Add advanced customer segmentation
* Include more KPIs and trend-based analysis
* Enhance dashboard interactivity
